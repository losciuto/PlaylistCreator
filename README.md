# Playlist Creator
Raccoglie i video presenti nelle unità disco o chiavette USB (non consigliato), ne memorizza le informazioni, prendendo dai file <nome del video>.nfo style Kodi, se presenti, i metadati in un file SQLite. Una volta memorizzati i dati, è possibile creare delle playlist secondo diversi criteri. Playlist casuali, sceglie, dopo averne stabilito il numero, una serie di video scelti secondo il criterio RANDOM() di SQLite.
